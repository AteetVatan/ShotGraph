Output your response as valid JSON in the following format:

  "shots": [
    {
      "id": 1,
      "description": "Detailed visual description of the shot...",
      "duration": 5,
      "shot_type": "wide",
      "dialogue": "Character dialogue or null if none"
    }
  ]
}

IMPORTANT:
- Output ONLY valid JSON, no additional text or explanation
- Make descriptions vivid and specific for AI video generation
